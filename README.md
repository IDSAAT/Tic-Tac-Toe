🎮 Tic-Tac-Toe Game

A classic Tic-Tac-Toe game with two modes: Player vs Player and Player vs AI. Includes score tracking, win highlighting, and a smart AI opponent.

📌 Live Demo

View Live Demo at ""

🎯 Features
1. Two Game Modes: Player vs Player or Player vs AI
2. Smart AI: Prioritizes winning, blocking, and strategic moves (center → corners)
3. Score Tracking: Tracks X wins, O wins, and draws
4. Win Highlighting: Winning cells are highlighted in yellow with animation
5. Game Status Display: Shows whose turn it is or win/draw result

🚀 How to Run Locally

1. Clone the repository:
bash
   git clone https://github.com/yourusername/Tic-Tac-Toe.git
2. Navigate to the project folder:
   cd Tic-Tac-Toe
3. Open index.html in your browser

🎮 How to Play
1. Choose Player vs Player or Player vs AI
2. Player X always goes first
3. Click any empty cell to place your mark
4. First to get 3 in a row wins
5. Click New Game to play again
6. Click Reset Scores to clear scoreboard
